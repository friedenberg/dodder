# key_strings

Pre-allocated catgut.String constants for common field names used throughout the codebase.

## Constants

- `Blob`, `Comment`, `Description`
- `Genre`, `ObjectId`, `Sigil`
- `Tag`, `Tai`, `Type`, `TypeLock`
- `ZZRepoPub`, `ZZSigMother`

## Purpose

Provides shared string constants to avoid repeated allocations and ensure consistency
in field naming across serialization, indexing, and display formatting.
