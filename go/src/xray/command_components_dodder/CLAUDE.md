# command_components_dodder

Dodder-specific command component factories for building command environments.

## Key Types

- `Env`: Factory for creating local environments from command requests
- `LocalWorkingCopy`: Factory for creating local repository instances
- `LocalWorkingCopyWithQueryGroup`: Combines repo with query group building
- `Remote`: Remote repository connection factory

## Features

- Environment creation with XDG layout support
- Local working copy initialization
- Query group construction with workspace integration
- Remote repository connection setup
