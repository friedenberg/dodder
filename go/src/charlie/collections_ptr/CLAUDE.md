# collections_ptr

Generic pointer-based collection types with key functions.

## Key Types

- `Set[T, TPtr]`: Generic set storing pointers with string key lookup
- `MutableSet[T, TPtr]`: Mutable variant with add/delete operations

## Features

- Generic pointer-element storage
- String key-based lookup
- Iterator sequences for keys, values, and pointers
- Gob serialization support
