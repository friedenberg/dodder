# options_print

Print formatting options for CLI output with overlay configuration.

## Key Types

- `Options`: Full print option set (abbreviations, colors, fields)
- `Overlay`: Nullable overlay for merging configurations
- `OverlayBox`, `OverlayAbbreviations`: Nested overlay types

## Features

- Abbreviation control for zettel/markl IDs
- Box formatting options (time, tags, types, description)
- Color and flush printing options
- Overlay-based configuration merging
- CLI flag integration
