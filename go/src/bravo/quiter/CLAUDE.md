# quiter

Iterator utilities for working with Go's iter package and error-aware sequences.

## Key Functions

- `MakeSeqErrorFromSeq`: Convert Seq to SeqError
- `CollectError`: Collect sequence elements with error handling
- `Chain`: Chain multiple sequences together
- `Merge`: Merge sequences with comparison function
- `Reduce`: Reduce sequence to single value

## Features

- Error-aware iteration patterns
- Async and sync serialization
- Debug printing for iterators
