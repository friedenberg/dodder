# remote_connection_types

Enum type for remote connection protocols.

## Key Types

- `Type`: Connection type enum (Native, Socket, URL, Stdio, SSH)

## Supported Types

- `TypeNative`: Direct native connection
- `TypeSocketUnix`: Unix socket connection
- `TypeUrl`: HTTP/HTTPS URL connection
- `TypeStdioLocal`: Local stdio connection
- `TypeStdioSSH`: SSH-based stdio connection
